# 小报童文章下载器 - 安装指南

## 快速安装

### 方法一：开发者模式安装（推荐）

1. **下载插件文件**
   - 下载 `xiaobot-downloader-v1.0.0.zip` 文件
   - 解压到任意文件夹（如：`D:\xiaobot-downloader`）

2. **打开Chrome扩展管理页面**
   - 在Chrome地址栏输入：`chrome://extensions/`
   - 或者：菜单 → 更多工具 → 扩展程序

3. **启用开发者模式**
   - 点击页面右上角的"开发者模式"开关
   - 确保开关处于开启状态

4. **加载插件**
   - 点击"加载已解压的扩展程序"按钮
   - 选择解压后的插件文件夹
   - 点击"选择文件夹"

5. **确认安装**
   - 插件应该出现在扩展程序列表中
   - 状态显示为"已启用"
   - 浏览器工具栏会出现插件图标

### 方法二：CRX文件安装

1. **生成CRX文件**
   - 在扩展程序页面点击"打包扩展程序"
   - 选择插件文件夹，点击"打包扩展程序"
   - 生成 `.crx` 文件

2. **安装CRX文件**
   - 将 `.crx` 文件拖拽到扩展程序页面
   - 点击"添加扩展程序"确认安装

## 使用前准备

### 1. 登录小报童账号
- 访问 [小报童官网](https://xiaobot.net)
- 使用微信扫码登录
- 确保能正常访问已购买的专栏

### 2. 验证插件安装
- 访问任意小报童专栏页面
- 点击浏览器工具栏中的插件图标
- 应该能看到插件弹出窗口

## 使用方法

### 基本使用流程

1. **打开专栏页面**
   ```
   访问：https://xiaobot.net/p/专栏名
   ```

2. **启动下载**
   - 点击插件图标
   - 在弹出窗口中点击"开始下载"
   - 等待自动抓取完成

3. **查看下载文件**
   - 文件保存在：`下载文件夹/xiaobot_articles/`
   - 每篇文章对应一个 `.md` 文件

### 高级功能

#### 自定义专栏URL
- 在插件弹出窗口的"专栏URL"输入框中
- 输入要下载的专栏地址
- 支持跨专栏下载

#### 清除数据
- 点击"清除数据"按钮
- 清空已抓取的文章缓存
- 重新开始抓取流程

## 常见问题解决

### Q1: 插件图标不显示
**解决方案：**
- 检查插件是否已启用
- 刷新浏览器页面
- 重新加载插件

### Q2: 无法下载文章
**可能原因：**
- 未登录小报童账号
- 网络连接不稳定
- 专栏需要付费购买

**解决方案：**
- 确认已登录小报童
- 检查网络连接
- 确认已购买目标专栏

### Q3: 下载的文章格式异常
**解决方案：**
- 检查原文章是否包含特殊格式
- 尝试重新下载
- 手动调整Markdown格式

### Q4: 下载速度慢
**原因：**
- 插件会模拟滚动加载所有文章
- 大型专栏需要更多时间

**优化建议：**
- 保持网络稳定
- 不要在下载过程中切换标签页
- 耐心等待完成

## 权限说明

插件需要以下权限：

| 权限 | 用途 | 安全性 |
|------|------|--------|
| `activeTab` | 访问当前标签页 | 仅当前页面 |
| `scripting` | 注入脚本 | 仅小报童域名 |
| `storage` | 存储设置 | 本地存储 |
| `downloads` | 下载文件 | 系统下载文件夹 |
| `webRequest` | 拦截请求 | 仅小报童API |
| `*://*.xiaobot.net/*` | 小报童域名 | 仅小报童网站 |

## 卸载方法

1. **临时禁用**
   - 在扩展程序页面关闭插件开关

2. **完全卸载**
   - 在扩展程序页面点击"移除"
   - 确认删除插件

3. **清理数据**
   - 删除下载的文章文件
   - 清除浏览器缓存（可选）
